# 🥗 Smart Food Safety & Nutrition Tracker Web App

## 📌 Overview
Smart Food Safety & Nutrition Tracker is a full-stack web application that helps users analyze packaged food products using image scanning or barcode input. It extracts nutritional information and provides a health score along with personalized dietary insights.

---

## 🚀 Features
- 📷 Scan food labels using image upload
- 🔢 Enter barcode manually for product search
- 🧠 Automatic nutrition analysis (calories, sugar, fat, protein, etc.)
- 💯 Health score generation (0–100)
- ⚠️ Health alerts for harmful ingredients
- 🥗 Personalized diet recommendations based on user profile
- 📊 Dashboard with nutrition statistics

---

## 🏗️ Tech Stack
**Frontend:**
- React.js
- Axios
- Recharts

**Backend:**
- Node.js
- Express.js

**Database:**
- MySQL / MongoDB

**APIs:**
- OpenFoodFacts API

---

## ⚙️ How It Works
1. User uploads food image or enters barcode  
2. Backend extracts product data  
3. Nutrition values are analyzed  
4. Health score is calculated  
5. Results are shown with alerts & advice  

---

## 📊 Output Example
- Health Score: 48/100  
- Status: ❌ Avoid This Food  
- Alerts: High Sugar, Ultra-Processed Food  
- Advice: Reduce sugar, increase fiber intake
